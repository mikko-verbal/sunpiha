# Sun Piha – Kiinteistönhuoltopalvelut

Ultra-premium cinematic website clone for Sun Piha Oy – property maintenance services in Helsinki metropolitan area.

## Tech Stack

- **Next.js 15** (App Router) + **React 19**
- **Tailwind CSS v4**
- **TypeScript**
- **Framer Motion** – cinematic animations
- **Lenis** – smooth scroll
- **shadcn/ui** – UI components

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route | Description |
|-------|-------------|
| `/` | Cinematic homepage |
| `/kiinteistonhuolto` | Property maintenance landing page |
| `/palvelut` | Services overview |
| `/palvelut/[slug]` | Individual service pages (9 services) |
| `/yritys` | About us |
| `/ajankohtaista` | News archive |
| `/ajankohtaista/[slug]` | News articles (5) |
| `/tarjouspyynto` | Quote request form |
| `/yhteystiedot` | Contact page |
| `/tilaa-vaihtolava` | Skip/dumpster order + pricing |
| `/tietosuojaseloste` | Privacy policy |

## Build

```bash
npm run build
npm start
```
